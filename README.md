# Tynkerbell's AI Scheduler — (Open Source)

**Tynkerbell's AI Scheduler — Your Personal AI Scheduler to Optimize Your Workflow!**

**PREREQUISITES**
1) Settings → paste your OpenAI API key → Save. You can get the OpenAI API key from: https://platform.openai.com/settings/organization/api-keys
   * FYI, check your OpenAI API Rate Limits Below. https://platform.openai.com/settings/organization/limits
3) Generate Schedule → enter/upload Context + Goals → Generate JSON → Preview.
4) Run Alarms → Start Alarms → interact with popups (NEXT / CHANGE TIME / STOP).
5) Smart AI Settings → toggle Save Progress, Reminders, Auto Reroll, etc.
6) Optional: Export schedule.json for manual cloud backup.
Time format: HH:MM-HH:MM (24-hr). Timezone in JSON defaults to KST.

**HOW TO RUN**
1) Install Python (one time)
Install Python 3.10+ from python.org.
On Windows, check “Add Python to PATH.”

2) Create a virtual environment & install deps (one time)
Windows

python -m venv .venv
call .venv\Scripts\activate
python -m pip install --upgrade pip
pip install -r requirements.txt

macOS / Linux

python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
pip install -r requirements.txt

3) Launch the app
Windows
python app.py

macOS / Linux
python3 app.py

A window titled “AI Scheduler — Personal Power Mode” opens.
Let me know if there are any issues.
